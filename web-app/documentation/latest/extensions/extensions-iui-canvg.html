iUI CanVG

iui-canvg.js is an extension for the iUI framework. It is currently EXPERIMENTAL. It uses CanVG to render SVG images into HTML5 canvas tags.

SVG support is not required, but canvas support is required. For info on how to use this extension see Instructions.

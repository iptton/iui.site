iUI Extensions & plugins

Extensions & plugins

Our goeal with iUI is to come with the lightest and simpliest to work with touchscreen devices web framework.
All additionnals plugins, official or third party, are available as external "extensions".

Here is the list of extensions shipped with iUI 0.5

Cache Manager
If you need a deep control of application offline mode & cache swipe, this could really help

iUI-CanVG
Use CanVG to render SVG images into HTML5 canvas tags. SVG support is NOT required, but canvas support is.

Extended Styles
This is a library of CSS sets (photo gallery, additionnal buttons & lists, ...

Google Analytics
Get all ajax calls taken in account in your reports

iUIScroll
iScroll implementation in iUI made easy

JIT loader
Just In Time - Get Javascript included and initialize only when needed

Reachability
Perform 'reachability' testing for hybrid webapps that are wrapped in the UIWebView Cocoa Class

System Infos
It gives system informations. It may also be useful for the webapp developer you are to detect screen resolution, navigation mode, device capabilites & browser version.

TbBMod
Designed to offer some flexibility to some of the default behaviors of iUI, namely page context controlling of toolbar button behavior.

Theme switcher
Change which iUI theme you want to be used on-the-fly.

Unloader
All AJAX-loaded fragments will be unload when they are navigated away from (back button)

Video Extension
Place HTML5 video in iUI pages
